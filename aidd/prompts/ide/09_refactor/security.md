# Security Refactor Prompt

## Goal

Identify and fix security vulnerabilities to strengthen codebase security.

## Rules

- Security scanning
- Vulnerability assessment
- OWASP guidelines
- Secure coding practices
- Follow OWASP top 10
- Validate all inputs
- Sanitize outputs
- Use secure functions
- Apply least privilege

## Process steps

1. Scan codebase for vulnerabilities
2. Check input validation mechanisms
3. Review authentication/authorization
4. Identify injection attack risks
5. Fix identified security issues
6. Add security test coverage
7. Update vulnerable dependencies
8. Document security measures
