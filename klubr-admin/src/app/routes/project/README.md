# Gestions des Projets

## Création d'un projet

Pour créer un projet, il suffit de cliquer sur le bouton `Créer un projet` dans la page d'accueil de l'application. Vous serez redirigé vers un formulaire de création de projet. Il vous suffira de remplir les champs requis et de cliquer sur le bouton `Créer` pour valider la création de votre projet.

Une modale de choix de type de projet s'affichera:

### Projet de type `Modèles de projets standards`

Options disponibles:

- Projet vièrge
- Si le profil est de type "Klubr" et qu'il est au moins "Leader", il pourra créer un modèle de projet standard
- Liste de l'ensemble des catégories de modèles de projets standards "tous sports confondus"
- Liste de l'ensemble des catégories de modèles de projets standards "par sport" (filtre sur le sport de Klub du profil courant)

### Modèles de projets des Klubs affiliés `Modèles de projets de Fédérations`

Si un Klub est affilié à une (ou des) fédération(s) ```klubr.klubrAffiliations```, il pourra choisir un modèle de projet standard de la fédération.

Options disponibles:

- Liste de l'ensemble des catégories de modèles de projets de la librairie de modèles de projets de la fédération

### Modèles de projets propres `Modèles de projets du Klub du profil courant`

Si un Klub a une (plusieurs) bibliothèque(s) de projets ```klubr.template_projects_libraries```, il aura accès à cette catégorie.
C'est le cas des Fédérations par exemple.

Options disponibles:

- Création d'un nouveau modèle de projet
- Liste de l'ensemble des catégories de modèles de projets de la librairie

## Modification d'un projet

Pour modifier un projet, il suffit de cliquer sur le bouton `Modifier` dans la page d'accueil de l'application. Vous serez redirigé vers un formulaire de modification de projet. Il vous suffira de remplir les champs requis et de cliquer sur le bouton `Modifier` pour valider la modification de votre projet.

## Suppression d'un projet

Pour supprimer un projet, il suffit de cliquer sur le bouton `Supprimer` dans la page d'accueil de l'application. Vous serez redirigé vers une page de confirmation de suppression de projet. Il vous suffira de cliquer sur le bouton `Supprimer` pour valider la suppression de votre projet.

## Visualisation d'un projet

Pour visualiser un projet, il suffit de cliquer sur le bouton `Visualiser` dans la page d'accueil de l'application. Vous serez redirigé vers une page de visualisation de projet. Vous pourrez y voir toutes les informations relatives à votre projet.
